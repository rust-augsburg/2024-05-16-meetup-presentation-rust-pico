# Pico Boards

**Microcontroller: RP2040**
- `RP` Raspberry
- `2` number of 
- `0` ARM-Architecture M0+
- `4` SRAM: 264KB SRAM

**W-Model: Infineon CYW43439**
- **Integrated Wi-Fi**: 802.11 b/g/n
- **Bluetooth**: BLE Support
